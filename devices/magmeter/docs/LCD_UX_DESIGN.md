# Water Meter LCD UX Design

## Overview

This document defines the complete UX for the AgSys Mag Meter 2.8" transflective TFT display (320x240, ST7789). The design prioritizes daylight readability and simple 5-button navigation.

## Hardware

- **Display**: 2.8" Transflective TFT, 320x240, ST7789 controller
- **Buttons**: 5 tactile switches (UP, DOWN, LEFT, RIGHT, SELECT)
- **Button Events**: Short press (<500ms), Long press (>1s)

---

## Display Power States

The display has three power states to prevent LCD burn-in and save power:

| State | Backlight | Display | Timeout |
|-------|-----------|---------|---------|
| **ACTIVE** | 100% | ON | â†’ DIM after 60s inactivity |
| **DIM** | 50% | ON | â†’ SLEEP after 30s more |
| **SLEEP** | OFF | OFF | Wake on button press |

**Wake Behavior:**
- **Short press while sleeping** â†’ Wake to MAIN screen (ACTIVE), no action
- **Long press while sleeping** â†’ Wake + show PIN entry (if locked) or MENU
- **Any press while dimmed** â†’ Wake to ACTIVE, process the action

**Menu Timeout:**
- If in any menu screen for 60s without input â†’ Return to MAIN (DIMMED)

**Alarm Override:**
- Display stays ACTIVE while an alarm is active (no dim/sleep)

**Design Notes:**
- Outer frame: 2px border, 8px corner radius, 3px padding
- Numbers fill available space (48pt flow, 28pt trend/avg, 40pt total)

- Units use "L/min" not "L/m" to avoid confusion with "milli"
- Flow bar has subtle shadow for depth
- Dividers are 1px thin lines
- **Long press** any button to access menu (prevents accidental entry)
